# Bsale_test aplicación de prueba 
Tienda online la cual despliega productos agrupados por categorias

#Conlleva 
Este es un ejemplo de un tienda online reproducible y listo para probar, en el cual se encuentra:

1. Productos agrupados en categorias
2. Buscador de opciones
3. Páginas individuales de categorias

Esto en un front end (datos a interfaz gráfica para ser vista por el usuario) y api rest (comunicación entre interfaces).
Creando una página de inicio (index.html) en la cual se encuentran los productos y categorias y otra 
para ver las diferentes categorias, creando un botón de busquedas (busqueda.php). 

#Aplicación creada machoritariamente con php tanto backend como frontend, aunque no decía eso en las instrucciones, se me hizo mas facil en este lenguaje.
